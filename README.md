## چۆنیەتی بەکارهێنان (Implementation & Usage)

ئەگەر ویستت ئەم فایلە دابەزێنە و بیخەیتە نێو فۆڵدەری `assets` یان `raw` ی پڕۆژەکەت، بەپێی تەکنەلۆژیاکە بەم شێوەیە بانگی بکە:

### ١. لە Flutter
فایلەکە بخەرە نێو `pubspec.yaml` لە بەشی `assets`:
```yaml
flutter:
  assets:
    - assets/سەحی بوخاری کوردی.json

```
پاشان لە کۆدەکەدا بانگی بکە:
```dart
import 'dart:convert';
import 'package:flutter/services.dart';

Future<void> loadHadith() async {
  final String response = await rootBundle.loadString('assets/سەحی بوخاری کوردی.json');
  final data = await json.decode(response);
  print(data);
}

```
### ٢. لە JavaScript / Node.js / React
```javascript
import HadithData from './سەحی بوخاری کوردی.json';

console.log(HadithData);

```
### ٣. لە Swift (iOS)
```swift
if let path = Bundle.main.path(forResource: "سەحی بوخاری کوردی", ofType: "json") {
    do {
        let data = try Data(contentsOf: URL(fileURLWithPath: path), options: .mappedIfSafe)
        let jsonResult = try JSONSerialization.jsonObject(with: data, options: .mutableLeaves)
        print(jsonResult)
    } catch {
        print("Error reading JSON: \(error)")
    }
}

```
### ٤. لە Java (Android)
```java
import org.json.JSONArray;
import java.io.InputStream;
import java.nio.charset.StandardCharsets;

try {
    InputStream is = getAssets().open("سەحی بوخاری کوردی.json");
    int size = is.available();
    byte[] buffer = new byte[size];
    is.read(buffer);
    is.close();
    String json = new String(buffer, StandardCharsets.UTF_8);
    JSONArray jsonArray = new JSONArray(json);
} catch (Exception e) {
    e.printStackTrace();
}

```
